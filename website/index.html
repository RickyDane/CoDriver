Native FTP Client Integration

Work directly on remote file systems as if they were local storage volumes. CoDriver has a fully native, high-performance FTP client built directly into the file explorer tree.

  • Zero External Dependencies Connect natively to remote FTP servers directly within the sidebar disks panel without any external CLI tools, FUSE layers, or third-party wrappers.
  • Secure Credentials Store and retrieve secure FTP connection settings natively in-memory with safety and high-fidelity local configurations.
  • Blazingly Fast Traversal Crawl, query, and stream remote paths instantly using optimized asynchronous multi-threading, providing instant sidebar disk updates.

High-Ratio Archive Suite

Compress files instantly or decompress deep directory structures without leaving your file explorer. CoDriver features a native compression suite tuned for high-performance ratios.

  • Multi-Format Compression Compress folders or collections natively to ZIP, Zstandard, Brotli, or the high-ratio custom Density formats.
  • Built-in Extraction Decompress ZIP, 7Z, RAR, TAR, GZIP, BZIP2, Zstandard, and Brotli archives natively directly inside active folders.
  • Zero Dependencies No external winrar or 7zip tools needed. All algorithms are compiled natively inside CoDriver's Rust core.

Dual-Provider AI Superpowers

Supercharge your daily workspace natively. CoDriver features extensive integrated support for both Google Gemini and OpenAI models to automate visual and structural workflows.

  • AI Upscaler & Creative Styling Right-click any image to run high-resolution AI Super-Resolution, apply artistic Creative Enhancement, or restyle image compositions with customized prompts (e.g. converting a sketch to a cyberpunk oil painting).
  • AI Semantic Directory Organizer Cluster, categorize, and sort chaotic directories automatically. CoDriver builds an optimal proposed folder tree via LLM analysis, completing bulk file movements securely upon your review.

Under the Hood: High-Concurrency Core
