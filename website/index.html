CoDriver is designed to handle directories containing hundreds of thousands of files without missing a beat. It accesses disk blocks natively and streams files directly to the UI.

  • Instant Navigation Just start typing in any folder to filter matching elements on-the-fly and find files in milliseconds.

Side-by-Side Dual-Pane

Built for efficiency, CoDriver lets you toggle a balanced Dual-Pane layout using a single button. Say goodbye to dragging and dropping files across multiple windows.

  • F5 Copy & Move Shortcuts Highlight items and press F5 to copy or Shift+F5 to move them straight into the opposite pane.
  • Focused Deep Search Use F8 to trigger a robust concurrent file search isolated to the active folder tree.
  • Multi-Rename Tool Batch rename directories or collections with custom regex patterns using Cmd/Ctrl + Shift + M.

Native FTP & SFTP

Work directly on remote file systems as if they were local storage modules. CoDriver has native FTP and SFTP client integration built directly into the explorer tree.

  • Native Connections Connect natively to remote FTP and SFTP servers directly within the disks panel with zero external dependencies.
  • Secure Credentials Log in with secure password entry or SSH private keys natively stored within your local client.
  • Fast Traversal Crawl and stream remote paths instantly using optimized asynchronous multi-threading, providing real-time sidebar nodes.

High-Ratio Archive Suite

Compress files instantly or decompress deep directory structures without leaving your file explorer. CoDriver features a native compression suite tuned for high-performance ratios.

  • Multi-Format Compression Compress items instantly to ZIP, high-density ZStandard, Brotli, or Density archives via context menus.
  • Built-in Extraction Decompress zip, rar, 7zip, and tar (.tar.gz, .tar.bz2) files cleanly directly inside active folders.
  • Zero Dependencies No external winrar or 7zip tools needed. All algorithms are compiled natively inside CoDriver's Rust core.

Gemini AI Image Upscaler

Supercharge your visual workflow natively. Right-click any image inside the explorer and select the Upscale panel to generate high-resolution details on-demand.

  • Gemini AI Super-Resolution Upscale low-resolution previews using integrated Google Gemini 3.1 Flash Image models, enhancing details and sharpening textures without pixelation.
  • Aspect Ratio & Model Controls Toggle between traditional algorithms (Lanczos, Bicubic) and neural upscale paths with customizable target sizes (2x, 4x) and filenames.
  • Secure Local Keys Your Google AI Studio API key is stored safely within your system's secure local configuration. No middleman proxies or cloud storage.

Under the Hood: High-Concurrency Core

CoDriver bypasses heavy file system indexing services, bringing frontend events down to metal operations in microseconds.

Modern UI (jQuery)

A GPU-accelerated web view renders our stunning, responsive UI, reacting in real-time to active states.

Tauri IPC Channel

Tauri v2 bridges UI interactions directly to Rust backend crates, maintaining thread safety and zero latency.

Rust Core & jwalk

Parallel disk walking via rayon and notify-driven file monitors react instantly to directory adjustments.

Local File System

Direct system interaction with Windows, macOS, and Linux kernels with no intermediary cache buffers.
